# 팝업은 단일 프로필 뷰 + 칩 스위처, 프로필 내부는 언더라인 탭

전 프로필 카드를 세로로 적층하던 팝업 구조를 버린다. 적층은 프로필 수에 비례해 스크롤 벽이 되고, 카드-in-카드 보더와 행마다 항시 노출되는 옵션 2줄이 겹쳐 "복잡한 UI"의 근원이었다. 대신:

- **팝업(420px)**: 상단 프로필 칩 행(on/off 도트 표시, 클릭=전환) + 본문은 선택된 프로필 하나만. 프로필 내부는 언더라인 탭(수정/필터)으로 분리 — 한 화면에 한 관심사.
- **행은 테이블형**: 컬럼 헤더 + 얇은 구분선. 행별 카드/보더 제거. 모드(덮어쓰기/덧붙이기)·주석 등 세부 옵션은 행 선택 시에만 확장.
- **탭 앱(넓은 표면)**: 아이콘 레일(프로필/백업/환경설정) + 사이드바(프로필 목록·검색) + 본문 단일 프로필. 팝업과 본문 컴포넌트를 공유한다.
- **테마**: 시스템 연동 다크+라이트. 구분은 보더 대신 명도·여백, 액센트는 1색.

트레이드오프: 전 프로필의 수정 내용을 한 화면에서 훑는 능력을 잃는다 — on/off 상태 파악은 칩 도트로 대체하고, 상세 비교가 필요하면 탭 앱을 쓴다. 프로필 우선순위(겹침 시 상단 우선)는 칩 순서가 그대로 표현한다.
