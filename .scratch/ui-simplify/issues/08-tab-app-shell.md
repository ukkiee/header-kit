# 08 — 탭 앱 셸 (아이콘 레일 + 사이드바 + 검색)

**What to build:** 넓은 표면(탭 앱)이 아이콘 레일(프로필/백업/환경설정) + 사이드바(프로필 목록·검색) + 본문 레이아웃이 된다. 사이드바에서 프로필을 고르면 본문이 전환되고, 검색으로 목록이 필터링된다. 본문 편집 컴포넌트는 팝업과 공유 — 표면을 오가도 편집 경험이 같다. 팝업 하단 접이 패널(백업/환경설정/전송)은 탭 앱에서 레일 화면으로 승격된다.
