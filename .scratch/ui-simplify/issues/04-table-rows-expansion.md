# 04 — 테이블형 행 + 선택 시 확장 (헤더/쿠키/Set-Cookie)

**What to build:** 수정 목록이 이름/값 컬럼 헤더 + 얇은 구분선의 테이블이 된다. 행은 기본 1줄(체크박스+종류 뱃지+이름+값)이고, 행을 선택하면 그 행만 모드(덮어쓰기/덧붙이기)·빈 값 처리·주석·확대 편집기가 확장된다(단일 확장 — 다른 행 선택 시 이전 행 접힘). 행별 카드/보더는 퇴역한다. 긴 값은 행 안에서 truncate된다.

**Blocked by:** 03 — 언더라인 탭 (행이 수정 탭 안에 산다).

**Status:** ready-for-agent
