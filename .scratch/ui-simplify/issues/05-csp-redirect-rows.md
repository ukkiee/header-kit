**What to build:** CSP와 리다이렉트 수정도 같은 테이블 행 언어로 보인다 — 1줄 요약(종류 뱃지 + 핵심 내용) + 선택 시 전용 편집 UI(CSP 디렉티브 목록, 리다이렉트 패턴/치환) 확장. 목록의 시각 언어가 수정 종류와 무관하게 일관된다.
